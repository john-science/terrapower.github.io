class armi.utils.dochelpers.PyReverse(name, arguments, options, content, lineno, content_offset, block_text, state, state_machine)[source]¶

Bases: docutils.parsers.rst.Directive

Runs pyreverse to generate UML for specified module name and options.

The directive accepts the same arguments as pyreverse, except you should not specify --project or -o (output format). These are automatically specified.

If you pass -c to this, the figure generated is forced to be the className.png like BurnMatrix.png. For .gitignore purposes, this is a pain. Thus, we auto-prefix ALL images generated by this directive with pyrev_.
