10. armicontrib.dif3d package¶

This is the package documentation for the ARMI-DIF3D plugin.

DIF3D is an ANL physics code that computes neutron or photon flux distributions in a nuclear reactor.

See ARMI-DIF3D Plugin for main documentation.
